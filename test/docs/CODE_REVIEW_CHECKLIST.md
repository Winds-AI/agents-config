## Required Checks

- [ ] Sequential-thinking pass was completed before coding.

- [ ] Naming is clear and consistent.
- [ ] Control flow is simple; invalid paths fail fast.
- [ ] No unnecessary abstraction or duplication.
- [ ] Tests cover edge cases identified during sequential thinking.
